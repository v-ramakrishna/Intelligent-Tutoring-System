Calculus

Calculus is the study of how things change.

Differential Calculus cuts something into small pieces to find how it changes.

Integral Calculus joins (integrates) the small pieces together to find how much there is.

Limits

Limits help determining the variable that they are approaching but never reach it.

Example:

f(x) = (x2 - 1) / (x - 1)

If x = 1,

f(x) = (12 - 1) / (1 - 1) = 0/0

Since 0/0 is an indeterminate value we must figure a way on how to solve this.

Example Continued:

f(x) = (x2 - 1) / (x - 1)

If x = .9

f(x) = (.92 - 1) / (.9 - 1) = 1.9

Example Continued:

f(x) = (x2 - 1) / (x - 1)

If x = .99,

f(x) = (.992 - 1) / (.99 - 1) = 1.99

Example Continued:

f(x) = (x2 - 1) / (x - 1)

If x = .999,

f(x) = (.9992 - 1) / (.999 - 1) = 1.999

As we get closer to 1, we are able to determine that x gets closer to 2. Although when we plug 1 into x we get 0, because x never gets there.

lim(x->1) = (x2 - 1) / (x - 1) = 2
